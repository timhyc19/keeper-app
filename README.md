# keeper-app
A post and delete note page made using react. Concepts taught from Udemy Web Dev Bootcamp.
Topics included:
- JSX and Babel
- Inline styling, components
- Importing and exporting modules
- React props, DevTools
- Map / Filter / Reduce, Arrow functions
- useState Hooks
- Component tree
